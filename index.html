A simple Pinterest API Test

This API test shows how to retrieve a user Pinterets profile boards with related image, title and link to the board. According to the official API documentation it is only possible to retrieve a 60x60 image which has been associated with each board.

Clicking "Authorize!" a popup will appear requesting access to the user data, this will grant the app access to his/her data which can be retrieve according to the API endpoints.

Necessary steps

In order to achive the result showed here, one needs to create an App inquiry on the Pinterest developer webiste. This will grant access to the App ID which is necessary in order to submit requests to the API.

The rest is well commented on the source code.
